Symfony Demo Api
========================

Installation
------------

```bash
$ git clone https://github.com/DimaKrit/Swivl-Symfony3-API.git

```

Usage
-----

```bash
@Rest\Get("/classroom")
<domain>/app_dev.php/classroom

----------
Rest\Get("/classroom/{id}")
<domain>/app_dev.php/classroom/<id>

----------
Rest\Post("/classroom/")
<domain>/app_dev.php/classroom/

----------
Rest\Put("/classroom/{id}/")
<domain>/app_dev.php/classroom/<id>/

---------
Rest\Delete("/classroom/{id}")
<domain>/app_dev.php/classroom/<id>

```

------------------
